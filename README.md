BMW Sales Dashboard

This project presents an interactive BMW Sales Dashboard built using Power BI to analyze sales performance across different vehicle models, regions, and product characteristics from 2010 to 2024.

The dashboard provides insights into revenue trends, sales volume, regional performance, fuel preferences, transmission types, and relationships between vehicle specifications and pricing, enabling a comprehensive understanding of BMW's market performance.

📊 Key Features : 
Sales Performance Overview ;
Total Units Sold ;
Total Revenue ;
Average Vehicle Price ;
Trend Analysis ;
Annual sales and revenue trends (2010–2024) ;
Regional Analysis ;
Sales distribution by region ;
Model Performance ;
Top-performing BMW models based on sales and revenue ;
Fuel Type Distribution ;
Transmission Type Distribution ;
Vehicle Classification Analysis ;
Color Preference Analysis ;
